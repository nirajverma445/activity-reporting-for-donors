## 📋 What is activity-reporting-for-donors?

activity-reporting-for-donors is a user-friendly dashboard designed to help nonprofit organizations create clear and professional donor reports. It combines key performance indicators (KPIs), impact stories, and powerful charts in one place. You can quickly generate stakeholder-ready presentations with a single click. This tool uses modern web technologies called React and TypeScript, but you don't need to know any coding to use it.

If you want to track donations, show progress, and share stories with your donors, this dashboard makes the process much simpler and faster.

## 📊 How to Use the Dashboard

This app is built to help you create clear reports without any hassle. Here’s a simple guide to get familiar:

### Adding Data

- Look for the "Add Data" button or menu option.
- Input your donation details like donor name, date, amount, and any notes.
- Save each entry. You can enter many donations over time.

### Viewing KPIs

- The dashboard shows KPIs such as total donations, donor retention rate, and average donation size.
- These numbers update automatically as you add new data.

### Exploring Impact Stories

- You have a section to add stories highlighting how donations made a difference.
- Upload text and images easily.
- These stories help keep donors engaged.

### Visualizing Data

- Charts such as bar graphs and pie charts display donation trends.
- Use filters to customize views by date, donor, or program.

### Exporting Reports

- When you are ready to share, click the "Export" button.
- Choose "One-Click PPTX Export" to create a PowerPoint file.
- This export contains charts, KPIs, and impact stories, ready for presentations.
